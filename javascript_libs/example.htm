Usage

Insert this line into your HTML body (not in the head section): 
<script language="JavaScript" src="tooltip.js"></script>
Use the toolTip function with mouse-over and mouse-out events:
  • To show a tooltip, use this syntax: toolTip(text, width, opacity)
    Note: width (pixels) and opacity (percent: 0 - 100) are optional. Opacity is not supported by all browsers.
  • To hide a tooltip, use this syntax: toolTip()

Example

<a href="#" onMouseOver="toolTip('Just a test', 150)" onMouseOut="toolTip()">some text here</a>
